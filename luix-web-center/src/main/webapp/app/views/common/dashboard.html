More and more extra great features

Simple, High performance, Intelligent, Powerful

LUI️✘

Interface-oriented RPC Call

Based on the TCP protocol and interface-oriented remote procedure call, the complex logic of the remote call is encapsulated, and the caller can complete the complex remote procedure call through the simplest local method call.

LUI️✘

Service Registration and Discovery

After the application is started, there is no need to do related network configuration, the application will automatically register its own IP to the registry. Any application under the registry can automatically perceive and discover all other application services.

LUI️✘

Fault Tolerance and Load Balancing

Built-in multiple service call fault tolerance strategies and client load balancing algorithms. Service call fault tolerance strategies include: failfast, failover, etc. Client load balancing algorithms include: random, consistent hashing, low-load party calls, etc.

LUI️✘

High-performance Protocols and Serializers

Based on the TCP long link, it is suitable for commonly used inter-service calls. Provides a variety of high-performance serialization and deserialization algorithms. The program can intelligently select the optimal serialization and deserialization algorithms according to different data types.

LUI️✘

Service Governance Control Center

Provides a powerful Web-based service management control center program, through which all services can be managed and monitored, and the relevant configuration parameters of the service can also be dynamically modified.

LUI️✘

Intelligent Configuration

The product design concept is powerful and easy to use. The system does not provide too many parameter configurations, but the program can select the optimal parameters based on intelligent judgment, freeing the programmer's brain.
